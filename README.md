### Component Property

#### tab-feature-item

**tow** properties

label: 标签名称

- **style**
  - width: 控制icon背景宽度
  - height: 控制icon背景高度
  - color: 控制字体颜色
  - fontSize: 控制字体大小
  - backGround: 控制icon背景颜色
  - iconName: 控制icon图标

#### banner-title

**tow** properties

- title: 标题名称
- typeColor: 背景颜色

#### box-show

**five** properties

- id: id
- name: 标题
- picUrl: 图片链接
- playCount: 播放次数
- copywriter: 编辑推荐?
- styles
  - width: 控制img宽度
  - height: 控制img高度

#### button-expand

**tow** properties

- content: 文字内容
- iconName: icon名称

#### play-count

**tow** properties

- count: 播放次数
- styles
  - fontSize: 字体大小
  - color: 字体颜色,
  - iconSize: icon大小
